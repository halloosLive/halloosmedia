# JZN Live Platform

This repository hosts the JZN Live Platform for GitHub Pages.

## Deploy
1. Push these files to your public repo (e.g., jzn-live).
2. Enable GitHub Pages: Settings → Pages → Deploy from a branch → main / root.
3. Open https://halloosMedia.github.io/jzn-live/ (replace repo name if different).

## Notes
- index.html contains Firebase compat SDK and RTDB integration.
